# Output Templates

Use these templates as a starting point. Adjust length and emphasis based on paper length and user request.

## Quick TL;DR (Chinese)

- **一句话总结**: ...
- **贡献要点** (3-6 条):
- **方法概览**: ...
- **关键结果** (带数字/对比):
- **限制/风险**: ...
- **值得追问的问题** (2-4 条):

## Deep Analysis (Chinese, default)

### 1. Paper Card
- 标题 / 作者 / 会议(年份)
- 任务与问题定义
- 贡献要点 (3-6 条)
- 数据集 / Baselines / 评价指标

### 2. 方法
- 直觉与动机
- 模型或算法结构
- 关键公式与符号解释
- 训练/推理流程与复杂度

### 3. 实验与结果
- 实验设置与对比
- 关键指标与主要表格结果
- 主要消融与敏感性分析

### 4. 结论与局限
- 主要结论
- 局限与潜在失败模式
- 适用范围与可能的负面影响

### 5. 复现要点
- 数据预处理与划分
- 超参/训练细节/随机种子
- 计算资源与训练时间
- 代码或模型发布情况

### 6. 重要问题
- 需要进一步验证的点
- 可能的改进方向

## Review-Style Critique (Chinese)

### Summary
- 2-4 句话概述目标与方法

### Strengths
- 3-5 条

### Weaknesses
- 3-5 条

### Questions
- 3-6 条（针对设置、消融、对比、可复现性）

### Recommendations
- 建议改进方向或补充实验

## Reproduction Notes (Chinese)

- **核心算法步骤** (伪代码或步骤列表)
- **必要实现细节** (损失函数、正则、归一化、采样策略)
- **关键超参数** (学习率、batch size、epoch、optimizer)
- **数据与预处理**
- **训练资源** (GPU/TPU、时长)
- **不确定/缺失信息** (显式列出)

## Optional: Section-by-Section Map

If the user wants full coverage, provide a section map:
- Section name -> 1-2 sentence summary -> key figure/table
